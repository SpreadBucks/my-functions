# My Functions

## What is My Functions?

My Functions is a calculator app with which users can define unique functions like "Calculate My Car's MPG", "Pithagorean Theorem", etc.

## How to run

Install ionic.
```
npm install -g ionic
```

Then, serve.
```
ioninc serve
```

Check from browser.
- http://localhost:8100
- http://localhost:8100/ionic-lab
